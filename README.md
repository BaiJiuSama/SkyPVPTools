## SkyPVPTools

是的你没看错，这只是我一时兴起做的小玩具\
但是缝合了部分功能，对开SkyPVP服务器的Developer有点帮助\
比如说 Fix、Sell和GameMode，都有一些小功能在内

### 如何使用?

实际上来说非常简单，你只需要进入到该项目的目录\
并且确保你已经配置好了Gradle环境\
就可以在项目目录下输入

```bash
./gradle build
```

或者，你可以在打开你的IDEA之后执行Shadow的shadowJar操作\
构建后的Jar位于 ```*/SkyPVPTools/build/libs``` 内\
这样你就可以获得我的无敌大神缝合插件了\

最后，祝你开心
